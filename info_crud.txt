1 - Cadastro de clientes:
nome, telefone, email, cpf(opcional), data de nascimento
historico de agendamemtos e atendimentos
observacoes(preferencias, restricoes, cor de tinta)


Ferramentas/comandos:
npm init -y
npm i express cors mysql2 nodemon dotenv sequelize sequelize-cli axios react-hook-form bcrypt jsonwebtoken
npx sequelize init

Iniciar front: npm run dev
Iniciar back: npm start

Front:

Paginas:
 
            <Route path='/' element={<Home />} /> | pagina inicial onde lista todos os clientes

            <Route path='/clientes' element={<Clientes />} /> | cadastro/forms de clientes

            <Route path='/cliente/:id' element={<Cliente />} /> | read de cliente individual

            <Route path='/login' element={<Login />} /> | login com autenticacao de token

            <Route path='/cliente/edit/:id' element={<ClienteEdit />} /> | forms de edicao de cliente

            <Route path='*' element={<Error />} />

Back:
.env -> login, senha e token
